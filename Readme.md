Portfolio
---

### Description

This is my portfolio, it's a simple static website created with Vite.js for React.js.
Using Atomic Design for the architecture and Sass modules for the style.

### Installation

```bash
npm install
```

### Usage

#### With NPM

```bash
npm run dev
```

#### With Docker

```bash
docker-compose up --build
```
